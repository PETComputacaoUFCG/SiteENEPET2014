Como Submeter

Os trabalhos deverão ser submetidos através do Portal PET, pelo tutor ou integrante do grupo. Para submeter é só fazer login no portal,clicar na aba Atividades/Eventos, selecionar XIII ENEPET - Campina Grande e submeter o artigo.
